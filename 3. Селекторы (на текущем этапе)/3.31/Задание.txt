Расскажите, какие свойства применятся к заголовкам и какие будут значения этих свойств:

<h2>text</h2>
h2 {
	font-size: 20px;
	color: blue;
}
h2 {
	font-style: italic;
	color: red;
}
-------------------------------
Будут следующие свойства для h2:

h2 {
    font-size: 20px;
	font-style: italic;
	color: red;
}
